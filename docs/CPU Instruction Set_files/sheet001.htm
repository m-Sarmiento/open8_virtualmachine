JSR 10111 111 - Jump to Subroutine (pushes the return address, then jumps to the address specified in subsequent bytes)

LDA R0-R7 11101 xxx N,Z Load Rn from Fixed Address (Address follows instruction)
LDX R0-R7(+) 11110 xxx N,Z,(C*) Load R0 from Address referenced by Rn+1:Rn (See CPU options for exact behavior)
LDO R0-R7(+) 11111 xxx N,Z,(C*) Load R0 from Address referenced by Rn+1:Rn plus an offset  (Offset is stored in subsequent byte) (See CPU options for exact behavior)
